# Pig Game

A simple dice game built with vanilla HTML, CSS, and JavaScript, inspired by the classic Pig Game rules.

## Game Rules

- Two players take turns

- On each turn, a player can:

  - Roll the dice
    - If the dice is 1, the turn ends and the current score is lost
    - Otherwise, the dice value is added to the current score

  - Hold
    - The current score is added to the total score
    - The turn switches to the other player

- The first player to reach the winning score wins the game

## Features

- Two-player gameplay

- Random dice rolls

- Turn-based logic

- Score tracking

- Restart game functionality

## How to Play

- Open index.html in your browser

- Click Roll Dice to play

- Click Hold to save your score

- First player to reach the winning score wins
